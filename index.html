Drawing on Canvas using JavaScript

All drawings on the HTML canvas must be done with JavaScript: Drawing Peppa Pig and his family as a static example of Canvas, it really was challenging , and time consuming considering the many co-ordinates that were plotted. Drawing the head function i used the Arc method to draw a circle,then set the start angles to 0 and end angle to 2*Math.PI.
Then i used false(default) and then used lineTo to joint the nose lines together. I used quite a lot of methods and properties in completing the canvas drawing , the initial problem i had was the conflicting issues between the gradient used for the sky and the sun, but was simply resolved when i put them all in functions, hence i have many functions in the code. The radii of the glow was set as follows:

  • InnerRadius = 1
  • OuterRadius = 50
The Radius of the entire circle was set at:
  • Radius = 60

Methods used in this project are as follows:

Paths , Quadratic Curves , Arcs , Lines , Bezier Curves , Rectangles , Radial Gradient , Linear Gradient , Compositing and finally GlobalAlpha.
